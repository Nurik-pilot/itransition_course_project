# Air Quality & Economic Data Pipeline (Microsoft Fabric)

## 📌 Описание проекта
Этот репозиторий содержит ноутбуки и пайплайны для загрузки, обработки и хранения данных в **Microsoft Fabric Lakehouse**.  
Проект демонстрирует архитектуру **Bronze → Silver** слоёв, где данные сначала собираются из внешних API, а затем очищаются и трансформируются для аналитики.

## 🗂️ Структура репозитория
- **Notebook1.ipynb** — сбор данных (Bronze слой):
  - Загружает данные о качестве воздуха из [OpenAQ API](ca://s?q=OpenAQ_API).
  - Получает данные о ВВП США из [World Bank API](ca://s?q=World_Bank_API).
  - Загружает валютные курсы USD/EUR из [ECB API](ca://s?q=ECB_API).
  - Сохраняет данные в таблицы:
    - `bronze_air_quality`
    - `bronze_gdp`
    - `bronze_fx_rates`

- **notebook_silver_transform.ipynb** — очистка и трансформация (Silver слой):
  - Обрабатывает данные такси Нью‑Йорка (`bronze_yellow_taxi` → `silver_yellow_taxi`).
  - Очищает и нормализует данные качества воздуха (`bronze_air_quality` → `silver_air_quality`).
  - Приводит к числовому виду данные ВВП (`bronze_gdp` → `silver_gdp`).
  - Преобразует валютные курсы (`bronze_fx_rates` → `silver_fx_rates`).

## ⚙️ Технологии
- **Microsoft Fabric** (Lakehouse, Spark, SQL)
- **PySpark** для обработки больших данных
- **Pandas** для промежуточных преобразований
- **REST API** для загрузки данных из внешних источников

## 📊 Архитектура пайплайна
1. **Bronze Layer** — сырые данные из API сохраняются без изменений.  
2. **Silver Layer** — данные очищаются, нормализуются и обогащаются метаданными (год, месяц, день недели, час).  
3. **Gold Layer (планируется)** — агрегированные метрики для аналитики и визуализации в Power BI.

## 🚀 Как использовать
1. Запусти `Notebook1.ipynb`, чтобы загрузить данные в Bronze слой.  
2. Запусти `notebook_silver_transform.ipynb`, чтобы преобразовать данные в Silver слой.  
3. Используй таблицы Silver слоя для построения аналитики и дашбордов.

## 📈 Возможные расширения
- Добавить Gold слой с агрегированными метриками.  
- Построить Power BI отчёты на основе Silver/Gold данных.  
- Подключить автоматизацию через Fabric Pipelines.
